# 程序打包说明

## 方法一：使用 build.bat（推荐）

1. 双击运行 `build.bat`
2. 等待打包完成
3. 可执行文件在 `dist` 目录中

## 方法二：使用命令行

### 1. 安装PyInstaller

```bash
pip install pyinstaller
```

### 2. 使用spec文件打包（推荐）

```bash
pyinstaller build.spec
```

### 3. 或直接使用命令打包

```bash
pyinstaller --name=CutImage --onefile --console --add-data="cut_image.py;." app.py
```

## 打包后的文件

- 可执行文件：`dist/CutImage.exe`
- 运行该exe文件即可启动程序

## 注意事项

1. **首次打包可能需要较长时间**（5-10分钟），因为需要收集所有依赖
2. **文件大小**：打包后的exe文件可能较大（100-200MB），因为包含了Python解释器和所有依赖
3. **杀毒软件**：某些杀毒软件可能会误报，这是正常现象
4. **运行环境**：打包后的exe可以在没有安装Python的Windows系统上运行

## 如果打包失败

1. 确保所有依赖都已安装：`pip install -r requirements.txt`
2. 确保在虚拟环境中：`venv\Scripts\activate`
3. 尝试清理之前的打包文件：删除 `build` 和 `dist` 目录
4. 检查错误信息并调整 `build.spec` 文件

## 测试打包后的程序

1. 进入 `dist` 目录
2. 双击运行 `CutImage.exe`
3. 程序会自动打开浏览器显示Web界面
